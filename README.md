# Sales-Dashboard-PowerBI
An interactive Power BI dashboard that analyzes key business metrics and visualizes trends using clear and dynamic charts. This project demonstrates data cleaning, analysis, and visualization skills to generate actionable insights from raw datasets.

Insights:
1. Created interactive dashboard to track and analyze online sales data
2. Used complex parameters to drill down in worksheet and customization using filters and slicers
3. Created connections, join new tables, calculations to manipulate data and enable user driven parameters for visualizations
4. Used different types of customized visualization (bar chart, pie chart, donut chart, clustered bar chart, scatter chart, line chart, area chart, map, slicers, etc)
